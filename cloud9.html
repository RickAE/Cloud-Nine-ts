Sobre

Cloud Nine Tecnologies School é uma escola de desenvolvimento de jogos, criada em Cotia - SP que ganhou o mundo, com a sua maneira inovadora de ensinar.

Acreditamos que o aprendizado está a cima de metodos ortodoxos e que programar é algo que todas as pessoas deveriam aprender.

Nossos professores são profissionais da área que estão sempre se renovando, para passar o conteúdo mais atual para os alunos.

Temos cursos das linguagens de programação mais modernas e mais usadas no mercado, sendo elas:

  • C#
  • Python
  • Java
  • Javascript
